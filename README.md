# Myntra-Sales-and-Product-Analysis--PowerBI
End-to-end Power BI project analyzing Myntra’s sales and product performance using a single dataset, featuring interactive dashboards for brand, category, customer, and regional insights.

##  Project Overview
This project presents an end-to-end Sales and Product Analysis for Myntra using Power BI. Both dashboards are built within a single Power BI report
using a combined dataset enabling consistent and integrated business insights.

The report contains:
- Product Analysis Dashboard
- Sales Analysis Dashboard

##  Business Objectives
- Analyze overall sales performance and revenue trends
- Identify top-performing brands and products
- Understand customer behavior across age groups
- Evaluate discount patterns across product sub-categories
- Compare sales performance by category, state, and time

## Tools & Technologies
- Power BI
- Power Query
- DAX
- CSV / Excel dataset

---

## Product Analysis Dashboard
### Key KPIs
- **Top Brand:** H&M
- **Top Product:** Jeans
- **Total Products:** 3,071

### Insights
- Bottomwear shows higher average discounts across sub-categories
- Majority of orders come from the **15–25 age group**
- Western wear and topwear dominate order volume
- Certain states contribute significantly to total order count
- Campus and Metro rank among top-rated brands

---

## Sales Analysis Dashboard
### Key KPIs
- **Total Sales:** 1.88M
- **Top Brand by Sales:** Puma

### Insights
- Puma leads overall sales contribution
- Sales trend shows a decline from 2021 to 2023
- Kids category records the highest average sales
- Telangana, Maharashtra, and Karnataka are top-performing states
- Infant and girls categories perform strongly within sub-categories
